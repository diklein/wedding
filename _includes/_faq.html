An FAQ? Why?

There's always so many questions when a wedding is approaching. This page will be updated to address the questions that we receive.

What is the wedding attire?

Cocktail attire. Suits for men and cocktail dresses for women. Women, please bring a pashmina as it may get chilly during dinner.

Will there be kids?

We love your babies and kids and teenagers. Really, we do. However, this will be a kid, baby, and teenager-free weekend.

Is there transportation?

All weekend festivities are walking distance within the town of Healdsburg. If you are staying outside of Healdsburg consider Lyft, a taxi, or a car service.

Who took the amazing photos on this website?

Kate Munsch! She's fantastic. https://www.katedocumentary.com

This website is so fast! How did you do it?

Well, it starts with using my own DigitalOcean SSD-based virtual server running NGINX. Then I drop a little Jekyll on that server to produce static HTML files. What does that mean? The server only needs to send over a few snippets of text to your device. Then I have all of the CSS placed in the head of each page, so you don't need to wait for an external file to load. Fewer server calls means faster load time. Lastly, I'm using imgix to optimize and deliver the photos. Neat, right?
